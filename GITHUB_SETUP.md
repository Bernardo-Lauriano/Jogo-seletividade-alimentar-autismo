# GitHub Pages Setup Instructions 🚀

## Passo a Passo para Publicar no GitHub

### 1️⃣ Criar uma Conta GitHub (se não tiver)
- Acesse: https://github.com
- Clique em "Sign up"
- Preencha seus dados

### 2️⃣ Criar um Novo Repositório
- Clique no ícone "+" no canto superior direito
- Selecione "New repository"
- **Nome do repositório:** `Joguinho` (ou outro nome)
- **Descrição:** "Um joguinho interativo para escolher o que comer!"
- ✅ Marque "Public" (importante para GitHub Pages)
- ✅ Marque "Add a README file"
- Clique em "Create repository"

### 3️⃣ Fazer Upload dos Arquivos
- No repositório, clique em "Add file" → "Upload files"
- Selecione os seguintes arquivos:
  - `index.html`
  - `styles.css`
  - `script.js`
  - `README.md` (já incluído)
- Clique em "Commit changes"

### 4️⃣ Ativar GitHub Pages
- Vá em "Settings" (aba no repositório)
- Procure por "GitHub Pages" (lado esquerdo)
- Em "Source", selecione: `main` branch
- A página será construída automaticamente
- Sua URL será: `https://seuuser.github.io/Joguinho`

### 5️⃣ Compartilhar
- Copie a URL gerada
- Compartilhe com amigos! 🎉

---

## ✨ Resultado Final
Seu jogo estará disponível em: `https://seu-usuario-github.github.io/Joguinho/`

Qualquer pessoa poderá jogar apenas acessando o link!
